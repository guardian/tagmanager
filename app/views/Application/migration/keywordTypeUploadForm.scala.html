@() TagManager - Update Keyword Types

Update Keyword Types for Tags

Upload a CSV file to update the keywordType for multiple tags at once. This will create a background job to process all the updates.

CSV Format

The CSV file should have four columns, but only id and keywordType are used.

Valid keywordType values are:

  • PERSON
  • ORGANISATION
  • EVENT
  • WORK_OF_ART_OR_PRODUCT
  • PLACE
  • OTHER

Example CSV:

id,webTitle,internalName,keywordType
education/ravensbournecollege,Ravensbourne,Ravensbourne College,PLACE
sport/nhl,NHL,NHL,ORGANISATION
sport/nick-kyrgios,Nick Kyrgios,Nick Kyrgios,PERSON
sport/olympic-torch,Olympic torch,Olympic torch,OTHER
stage/a-taste-of-honey,A Taste of Honey,A Taste of Honey,WORK_OF_ART_OR_PRODUCT
stage/adelaide-fringe-2015,Adelaide fringe 2015,Adelaide fringe 2015,EVENT


Note: After submitting, check the status page to monitor job progress.
